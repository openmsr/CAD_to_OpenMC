Welcome to The documentation for CAD_to_OpenMC
==============================================
**CAD_to_OpenMC** is a python tool, made to ease the process of
creating OpenMC-ready surface meshes from a CAD-generated representationof the geometry
in the form of a .step (or .stp)-file.

Check out the :doc:`source/usage` section for further information, including
:doc:`source/install`.

.. note::

   This project (and its documentation)is under active development.
